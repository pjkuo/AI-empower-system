# AI 專題生成平台

## 專案簡介

AI 專題生成平台是一套專為學生、教師與專題展示所設計的 AI 教學工具。

本平台可協助使用者：

- 將心智圖轉換為影片腳本
- 將 IPO 架構轉換為 AI Prompt
- 將 Python 專題轉換為 AI 短影音
- 自動生成旁白、字幕與分鏡
- 快速開啟 AI 影片生成網站
- 建立 AI 專題展示影片

平台適合：

- 資訊科技導論
- AI 應用課程
- STEM 教學
- 專題製作
- 數位媒體設計
- 智慧農業展示

# 常見問題 FAQ

## Q1：影片生成失敗？

### 原因

- 免費額度不足
- Prompt 過長
- AI 平台繁忙

### 解法

- 改用英文 Prompt
- 縮短內容
- 重新整理後再試

---

## Q2：為什麼有浮水印？

免費版通常會加入浮水印。

### 建議

可使用：

- CapCut
- 字幕
- 校名 Logo

進行遮蓋。

---

## Q3：影片太短？

建議：

多段生成後再進行剪輯。

---

## Q4：旁白不自然？

建議：

- 降低語速
- 使用柔和語音
- 增加停頓
