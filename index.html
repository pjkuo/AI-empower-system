    ⚙ AI 後端設定
    
    
    
      支援 10+ AI 提供者。金鑰僅儲存於瀏覽器本地，不會上傳任何伺服器。
    
      新功能：可設定備援金鑰池，主金鑰額度用盡時自動切換。
    
    

    
    
      
      
      
    
    

    
    
    
      
    
        
        
      
    
      
    
        
        
        
    
      
    
      
    
        
        
        
    
      
    
      
    
        
        
        
    
          在 Cloudflare Dashboard 右側可找到
        
    
      
    
      
    
        
        
      
    
    
    

    
    
    
      
    
        運作原理：當主金鑰回傳 quota / rate-limit 錯誤時，系統會依序嘗試備援金鑰，直到成功為止。
      
    
      
      
    
        
        
    
        
    
          
          
        
    
        
        
        
      
    
    
    

    
    
    
      
    
        一鍵測試所有已設定的金鑰，找出哪些可用、哪些超出額度。
      
    
      
    
        
      
    
      
    

      
    📖 常見錯誤對照
    
      
    
        
    錯誤關鍵字意思解法
    
        
          
    quota exceeded免費額度用完等 24 小時 或 切換備援
    
          
    rate limit / 429太頻繁等 60 秒 或 切換備援
    
          
    invalid api key / 401金鑰錯誤重新複製金鑰
    
          
    model not found / 404模型名錯清空「模型名稱」用預設
    
          
    CORS / fetch failed跨域被擋該提供者不支援瀏覽器直連
